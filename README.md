# E-commerce Data Pipeline

##  Project Overview
This project is an end-to-end ETL (Extract, Transform, Load) pipeline built using Python and SQL.

## Tech Stack
- Python
- Pandas
- SQLite
- SQLAlchemy

## Workflow
1. Extract data from CSV files
2. Transform and clean the data
3. Load data into SQLite database
4. Run SQL queries to verify

##  Project Structure
- data/raw → Raw datasets
- data/processed → Cleaned data
- scripts → ETL scripts
- ecommerce.db → Database file

##  Output
- Clean dataset stored in CSV
- Structured data stored in database
