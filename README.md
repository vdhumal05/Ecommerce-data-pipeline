 E-commerce ETL Pipeline

 Overview

Built an end-to-end ETL pipeline using Python to process e-commerce data.
The pipeline extracts raw data, cleans & transforms it, and loads it into a SQL database.

---

 Tech Stack

- Python
- Pandas
- SQLite
- SQLAlchemy

---

 Workflow

1. Extract – Load data from CSV files
2. Transform – Clean, handle missing values, merge datasets
3. Load – Store data into SQLite database

---

How to Run

python etl_process.py

---

 Output

- Cleaned data → "data/processed/cleaned_data.csv"
- Database → "ecommerce.db"

---

 Key Features

- Modular ETL design
- Data cleaning & transformation
- SQL database integration
